# Please_WebProject
💻 다양한 재능을 거래할 수 있는 마켓 서비스 제공

---

💻 사용 기술
- 개발 언어 : Java11, Javascript
- 프론트 : HTML5, CSS(Bootstrap), Jquery
- 서버 : Apache Tomcat 9.0
- DB : MariaDB 8.0.27
- 개발 도구 : MySQL Workbench, IntelliJ
- 프레임워크 : Spring Boot 2.7.0

---

📜 주요 기능

- 회원 관련
  - 회원 가입
  - 정보 수정
  - 회원 탈퇴
  - 장바구니
  - 구매 목록

- 제품 관련

  - 제품 등록
  - 제품 정보 수정
  - 제품 삭제
  - 리뷰 등록
